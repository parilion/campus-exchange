# 开发经验记录

## 踩坑记录

### 2026-02-13 | 登录失败问题
- **问题**：预先插入的用户密码哈希不正确
- **原因**：手动插入的 BCrypt 哈希与密码不匹配
- **解决**：通过注册 API 创建用户获取正确哈希
- **预防**：测试账号必须通过注册 API 创建，禁止手动插入密码

### 2026-02-13 | 数据库 DDL 操作受限
- **问题**：MySQL MCP 无法执行 ALTER TABLE 语句添加新字段
- **原因**：MCP 服务器配置禁止 DDL 操作
- **解决**：更新 SQL schema 文件，待手动执行或联系管理员开放权限
- **预防**：在实现新功能前先确认 MCP 权限，必要时准备手动执行的 SQL
